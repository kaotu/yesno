# yesno
## Create virtual environment `pipenv` OR `venv`

**VENV**

`$ python -m venv myenv`</br>
`$ source myenv/bin/activate`</br>
`$ pip install -r requirements.txt`</br>

**pipenv**

`$ pipenv install`</br>
`$ pipenv shell`

## How to Runserver
#### Must be at directory have manage.py file
`$ python manage.py runserver`
